    Глава 3. Решение:

    Варианты:

    • Svg
    • Canvas
    • Custom highlighter API

    Svg

    • + знакомо
    • + может вести себя как текст

    • - DOM не меньше
    • - позиционирование каждой буквы
    • - не адаптивно

    Canvas

    • + быстро

    • - вообще не текст

    Custom highlighter API

    • + нативная реализация
    • + чистый css
    • + уменьшение DOM
    • + ведет себя как текст
